# newScraper
A web app that scrapes articles from New York Times and lets users to view and leave comments on the latest news.

[Check it out](https://vast-plateau-65371.herokuapp.com/)

## Technologies Used
- Node.js
- Express.js
- MongoDB
- Mongoose
- npm packages
    - body-parser
    - express
    - express-handlebars
    - mongoose
    - cheerio
    - request
